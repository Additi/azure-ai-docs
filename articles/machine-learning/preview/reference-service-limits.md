# Service Limits

- Max allowed project folder size: 25 MB
    >[!Note]
    >This doesn't include `.git`, `docs` and `outputs` folder.

- Max allowed experiment execution time: 7 days
- Max size of tracked file in `outputs` folder after a run: 512 MB
